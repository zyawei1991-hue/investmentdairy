# 投资日报自动化系统 - 项目总览

## 📋 项目简介
**投资日报自动化系统**是一个价值投资者专用的自动日报生成与推送平台，支持A股市场的数据采集、盈亏分析、AI建议生成和多渠道推送。

## 🚀 核心功能
1. **⏰ 定时日报生成**：每日8:00开盘前简报 + 15:30收盘后详报
2. **📊 数据分析**：持仓盈亏、估值温度、市场情绪、资金流向
3. **🤖 AI智能分析**：Claude Opus模型生成投资建议
4. **📱 多渠道推送**：飞书群聊自动推送
5. **📄 文档存档**：腾讯文档详报存储
6. **📊 数据仓库**：腾讯智能表格历史数据存储
7. **📈 趋势分析基础**：多维表为历史分析提供数据基础

## 🔧 技术栈
- **后端语言**：Python 3.12.8
- **主要依赖**：requests, schedule, PyYAML
- **AI模型**：Claude Opus 4.5 (通过API调用)
- **API对接**：
  - 腾讯财经API (实时行情)
  - 东方财富API (备用)
  - 飞书OpenAPI (推送)
  - 腾讯文档MCP API (智能表格)
- **定时任务**：WorkBuddy自动化 & schedule库
- **数据存储**：
  - 智能表格：历史数据归档
  - 腾讯文档：日报文档存档
  - 本地config.yaml：配置管理

## 📊 核心功能详解

### 1. 持仓盈亏分析
- **多数据源**：腾讯财经 + 东方财富（备用）
- **精确计算**：考虑分红、拆分、历史成本
- **多维度统计**：总市值、当日盈亏、总盈亏、仓位比例

### 2. 市场数据采集
- **指数监控**：上证、深证、创业板指
- **资金流向**：北向资金净流入
- **市场情绪**：涨跌家数统计
- **估值温度**：主要指数PE分析

### 3. AI投资建议
- **模型选择**：Claude Opus 4.5
- **个性化分析**：基于持仓和市场数据
- **内容生成**：总结市场趋势 + 持仓分析 + 操作建议

### 4. 多渠道推送
- **⏰ 时间点**：每日8:00 + 15:30
- **📱 飞书推送**：简报到群聊（支持markdown格式）
- **📄 腾讯文档**：详报到智能文档（完整分析）
- **📊 智能表格**：数据存储到多维表（历史归档）

## 🛠️ 部署与运行

1. **automation**：投资日报详报-开盘前 (08:00, 使用 --evening)
2. **automation-2**：投资日报详报-收盘后 (15:30, 使用 --evening)
3. **automation-3**：投资日报手动推送 (08:05, 手动操作)
