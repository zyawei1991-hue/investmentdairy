## 📋 字段添加详细指引

### 1. 添加「日期」字段
```
操作：点击「添加字段」→ 名称输入「日期」→ 类型选「日期」→ 保存
注意：确保日期格式是"年-月-日"
```

### 2. 添加「报告类型」字段
```
操作：添加字段 → 名称「报告类型」→ 类型选「单选」
选项配置：
- 添加第一个选项：「开盘前」→ 设置颜色为蓝色
- 添加第二个选项：「收盘后」→ 设置颜色为绿色
```

### 3. 添加「总市值」字段
```
操作：添加字段 → 名称「总市值」→ 类型选「数字」
格式配置：
- 数字格式：货币
- 小数位数：2位
- 货币符号：人民币(¥)
```

### 4. 添加「当日盈亏」字段
```
操作：添加字段 → 名称「当日盈亏」→ 类型选「数字」
格式配置：
- 数字格式：货币
- 小数位数：2位
```

### 5. 添加「仓位比例」字段
```
操作：添加字段 → 名称「仓位比例」→ 类型选「数字」
格式配置：
- 数字格式：百分比
- 小数位数：1位
```
