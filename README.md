# 投资日报自动化系统

**价值投资者专用的自动日报生成与推送平台**

支持A股市场的数据采集、盈亏分析、AI建议生成和多渠道推送。

---

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| ⏰ **定时日报** | 每日8:00开盘前简报 + 15:30收盘后详报 |
| 📊 **数据分析** | 持仓盈亏、估值温度、市场情绪、资金流向 |
| 🤖 **AI智能分析** | Claude Opus模型生成投资建议 |
| 📱 **多渠道推送** | 飞书群聊自动推送 |
| 📄 **文档存档** | 飞书云文档详报存储 |
| 📈 **数据仓库** | 飞书多维表格历史数据存储 |

## 🚀 快速开始

### 1️⃣ 安装依赖

```bash
pip install -r requirements.txt
```

### 2️⃣ 配置文件

复制配置模板：

```bash
cp config_template.yaml config.yaml
```

编辑 `config.yaml` 文件：

```yaml
# 飞书配置
feishu:
  app_id: "your_app_id"
  app_secret: "your_app_secret"
  chat_id: "your_chat_id"

# 飞书多维表格配置
base:
  app_token: "your_base_token"
  table_id: "your_table_id"

# 持仓配置
holdings:
  - code: "600519"   # 股票代码
    name: "贵州茅台"   # 股票名称
    shares: 100       # 持仓股数
    cost: 1800.00     # 持仓成本价

# AI配置（可选）
ai:
  api_key: "your_api_key"
  model: "claude-opus-4-5"
```

### 3️⃣ 运行程序

```bash
# 单次运行
python main.py --morning     # 开盘前简报
python main.py --evening     # 收盘后详报

# 定时服务
python main.py               # 启动定时服务
```

## 📊 核心功能详解

### 1. 持仓盈亏分析

- **多数据源**：腾讯财经 + 东方财富（备用）
- **精确计算**：考虑分红、拆分、历史成本
- **多维度统计**：总市值、当日盈亏、总盈亏、仓位比例

### 2. 市场数据采集

- **指数监控**：上证、深证、创业板指、科创50
- **资金流向**：北向资金净流入
- **市场情绪**：涨跌家数统计
- **板块热点**：领涨领跌板块

### 3. AI投资建议

- **模型选择**：Claude Opus 4.5
- **个性化分析**：基于持仓和市场数据
- **内容生成**：市场趋势 + 持仓分析 + 操作建议

### 4. 多渠道推送

- **⏰ 时间点**：每日8:00 + 15:30
- **📱 飞书推送**：简报到群聊
- **📄 飞书云文档**：详报到云文档
- **📊 多维表格**：数据存储到多维表

## ⚠️ 注意事项

1. **API凭证安全**：不要将 `config.yaml` 提交到公开仓库
2. **飞书权限**：确保飞书应用有群聊推送权限
3. **数据延迟**：行情数据可能有几分钟延迟
4. **休市判断**：程序会自动跳过周末和节假日

## 🐛 常见问题

**Q: 飞书推送不显示？**
A: 检查机器人是否在群聊中，chat_id是否正确

**Q: 股票价格获取失败？**
A: 程序已内置重试机制，如持续失败请检查网络

**Q: 如何修改定时时间？**
A: 编辑 config.yaml 中的 schedule_time 参数

**Q: 如何添加新的持仓？**
A: 编辑 config.yaml 中的 holdings 列表
